    Saferides Application


    Founded in 2003, SafeRides is an entirely student-run FEB organization that offers free and safe rides home to JMU students on Friday and Saturday nights from 10PM-3AM.


    The goal for this project is to design, implement, test, and maintain an application for the Phone Operators, Log Operators and Navigators who run SafeRides to manage the ride requests and status of all the driver teams on a night of operations.


    In this project, I aim to primarily develop the back-end of the application. Key components include:

    1. Setup and administration of a relational SQL database to store ride and driver team information
    2. Design of the database tables and relationships
    3. Development of the code that retrieves pertinent information from and updates information in the database

    Phi Mu Alpha Website


    Founded in 1969, the Gamma Alpha Chapter of Phi Mu Alpha Sinfonia is a national music fraternity whose object is the development of the best and truest fraternal spirit; the mutual welfare and brotherhood of musical students; the advancement of music in America and a loyalty to the Alma Mater.


    I am currently working on the development of a website for the Gamma Alpha Chapter of Phi Mu Alpha Sinfonia. This website will provide general information about the fraternity, pictures from latest events, and a directory of current collegiate members of the Gamma Alpha Chapter.


    In this project, I am developing the front-end of the website. Since this is a simple website project, there is no database being used, though I am currently storing brother directory information in a Google Sheet where up to date information is fetched.

    JM'Uvies


    As a part of the Software Engineering class I took at JMU, my team and I were tasked to create a movie review based social media network. Using HTML, CSS, and the p5.js JavaScript library, our team created this social media network over 6 weeks using the SCRUM framework.


    In the development of this project, I primarily worked on:

    1. Movie review post creation
    2. Storage and retrieval of post information using a lightweight key/value storage solution built with Heroku
    3. Retrieval of up to date movie descriptions and ratings using The Movie Database (TMD) API
