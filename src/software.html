Software Engineering and Application Development

When I was in elementary school, I dreamed about being an inventor. I started learning more about coding and software through a JavaScript animation course on Khan Academy and by toying around with Lua in Roblox Studio. I learned more about the hardware behind computers by building and troubleshooting my own PC, and I was hooked. I knew that I wanted to work with technology when I grew up, and when I learned about the subject of Computer Science, that’s where I set my sights.


Through my university’s CS curriculum, I took a Software Engineering class where I learned about the software development lifecycle, working with a team of developers, working with an AGILE framework, and presenting the final iteration of our product. This class opened my eyes to the joy of collaborative development and thoughtful design before the implementation of a product, as well as how an iterative prototyping based approach to software development can tremendously aid innovation.

My Projects

SARR Tournament Manager

As a part of the Application Development class I took at JMU, my team and I designed and developed a client side tournament manager application that allows users to create athletic tournaments, manage teams, games, and venues, and get up to date weather and bracket information. This project was developed using JavaScript, HTML, and a mix of Bootstrap and custom CSS. In the development of this project, I primarily worked on:

  • Retrieval of tournament data from browser local storage and display of detailed information to users
  • Display of tournament bracket with Bracketry.js library
  • Integration with Google Maps API to display location of game venues to users

JM'Uvies

As a part of the Software Engineering class I took at JMU, my team and I were tasked to create a movie review based social media network. Using HTML, CSS, and the p5.js JavaScript library, our team created this social media network over 6 weeks using the SCRUM framework. In the development of this project, I primarily worked on:

  • Movie review post creation
  • Storage and retrieval of post information using a lightweight key/value storage solution built with Heroku
  • Retrieval of up to date movie descriptions and ratings using The Movie Database (TMDB) API
